# Поблочный разбор страниц detskiydomuss.ru

Источник: структура и тексты с [официального сайта](https://detskiydomuss.ru/); раздел новостей — по странице [Новости](https://detskiydomuss.ru/news/). (Browser MCP в сессии не ответил — разбор по контенту страниц.)

**Общий каркас на всех перечисленных URL:** верхняя полоса (ОК, VK, Telegram, «Написать нам», график, город) → полное меню (включая вложенное «Сведения о образовательной организации» с якорями на `/info#…`) → область `main` → подвал («Основные разделы», «Контакты», копирайт, ссылка на bus.gov.ru) → баннер cookie → ссылка «Version for visually impaired».

Ниже — **только уникальное содержимое `main`** по каждому адресу.

## Примечание по URL

Часть ссылок в меню ведёт на вариант **без** завершающего слэша (`/about`), сайт открывает тот же контент с редиректом на **`/about/`** и т.п. — для закладок удобнее канон с `/` в конце пути.
